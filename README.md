# Simple Dev Utilities

A growing collection of small Windows utilities built to scratch everyday developer itches.

Each utility lives in its own subfolder with its own README. They share this repository because they are small, focused, and not worth individual repos.

---

## Utilities

| Utility | What it does |
| --- | --- |
| [Explorer Orchestrator](ExplorerOrchestrator/README.md) | Manage a curated folder list and open all of them as tabs in a single Windows Explorer window |

---

## Stack

- **WPF / .NET 10** — UI and application logic
- **AutoHotkey v2** — Windows automation where WPF cannot reach

---

## Solution

`NelCapeTown.SimpleDevUtilities.slnx` — open this in Visual Studio or Rider to work across all utilities at once.
